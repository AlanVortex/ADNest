# ADNest

ADNest is a private financial management system designed for two users.

The application helps a couple manage their finances together by tracking expenses, organizing savings goals, and understanding spending habits through a simple and secure dashboard.

The name **ADNest** comes from the initials **A + D**, representing a shared financial space where both users build and manage their financial future.

---

# Purpose

ADNest is designed to help two users:

- track personal expenses
- track shared expenses
- create and monitor savings goals
- manage monthly budgets
- visualize financial progress

The goal is to provide a **clear and private financial overview** for better financial decisions.

# Core Features (Planned)

### Expense Tracking

Users can record:

- personal expenses
- shared expenses
- categorized spending

---

### Savings Goals

Users can create shared savings goals such as:

- vacations
- emergency funds
- large purchases

Each goal tracks progress toward a target amount.

---

### Budget Management

Users can define monthly budgets for categories such as:

- food
- transportation
- entertainment
- subscriptions

---

### Financial Dashboard

The dashboard will provide:

- monthly expense overview
- spending by category
- savings progress
- recent transactions
